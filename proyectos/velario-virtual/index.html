Velario virtual. Enciende tu vela.

Aplicación web online desarrollada tanto backend como frontend en PHP, Mysql y Javacript de velarío virtual. Permite encender una vela virtual “para que se cumplan todos tus deseos, peticiones o plegarias”. Características:

  1. Panel de administración con funcionalidades:
  • Modificar, eliminar velas creadas.
  • Crear, modificar, eliminar campañas de velas concretas (Ej. “Tiempo de Pascua”), configurar imágenes personalizadas de veas, tiempos de consumo de vela e inicio y fin de campaña.
  1. Página pública:
  • Crear vela gratuita (tiempo de consumo por defecto: 5 días).
  • Crear vela eterna (realizando una aportación económica - Integrada pasarela de pago Stripe).
  • Crear una vela de campaña si está habilitada (tiempo de consumo: el configurado en panel de administración).
  • Subir velas a primeras posiciones (re-encender vela apagada).
  • Posibilidad de enviar enlace de vela creada mediante e-mail a otra persona.

English:

Online web application developed (backend and frontend) using PHP, Mysql and Javascript. It allows you to light a virtual candle “so that all your wishes, requests or prayers may be fulfilled”. main features:

  1. Administration panel with features:
  • Modify and delete created candles.
  • Create, modify, delete specific candle campaigns (eg “All Saints' Day”), configure personalized candle images, candle consumption periods and start/end of campaigns.
  1. Public page:
  • Create free candle (default consumption period: 5 days).
  • Create eternal candle (by making an economic contribution - Stripe payment gateway integrated).
  • Create a campaign candle if it is enabled (consumption period: configured in the administration panel).
  • Move candles to first position listing (re-light extinguished candle).
  • Possibility of sending by email a candle link created to another person.
