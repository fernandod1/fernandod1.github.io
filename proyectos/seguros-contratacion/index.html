Gestión de contratación de seguro de salud

Aplicación web online desarrollada PHP, Mysql utilizando el framework Laravel. Permite gestionar el seguimiento del proceso de contratación de seguro de salud. Características:

  1. Panel de administración con funcionalidades:
  • Perfiles de administradores, agentes y clientes con permisos diferenciados de operativa en el panel.
  • Administrador tiene permisos para: Añadir, modificar, eliminar agentes, aseguradoras, clientes, médicos/laboratorios y sucursales.
  • Administrador tiene permisos para: Añadir, modificar, eliminar pólizas a clientes, tipos de pólizas, tipos de estados, tipos de checklists médicos a realizar sobre un nuevo cliente.
  • Gestión de documentación requerida en el proceso de contratación. Cambiar estado de documentos y opción de añadir comentarios.
  • Agentes tienen permiso para modificar sus datos de cuenta, consultar el estado del proceso de alta de nuevo cliente asociado a su cuenta de agente. Recibirá notificaciones mediante email de los cambios de estado de proceso.
  • Módulo de bonos para agentes. Permite motivar su trabajo con ganancias extras en función del total de pólizas tramitadas cada mes.
  • Módulo de historial de operaciones realizadas sobre cada contrato de cliente (libro de bitácoras).
  • Módulo de subida y almacenaje de documentos online de cada contrato o cliente.
  • Módulo de solicitud a cliente de pruebas médicas adicionales para completar el proceso de contratación.
English:

Online web application developed in PHP, Mysql using the Laravel framework. Allows you to manage the monitoring of a health insurance booking process. Features:

  1. Administration panel:
  • Profiles of administrators, agents and clients with different operating permissions.
  • Administrator has permissions to: Add, modify, delete agents, insurances companies, clients, doctors/laboratories and offices.
  • Administrator has permissions to: Add, modify, delete booking requests of clients, types of insurantes, types of states, types of medical checklists to perform on a new client.
  • Management of documentation required in booking process. Change document status and option to add comments.
  • Agents have permission to modify their account data, check the status of the new client booking process associated with their agent account. Agent will receive email notifications of process status changes.
  • Bonus module for agents. It allows to motivate work with extra earnings based on the total number of booking processed each month.
  • History module of operations carried out on each client booking (log system).
  • Module for uploading and storing online documents for each booking or client.
  • Client request module for additional medical tests to complete booking process.
