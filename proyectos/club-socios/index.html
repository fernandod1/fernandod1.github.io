Gestión recibos de pagos de socios.

Aplicación web online desarrollada PHP, Mysql utilizando el framework Laravel. Permite gestionar el historial de pagos y generar nuevos recibos recurrentes y de eventos de un club de socios. Características:

  1. Panel de administración con funcionalidades:
  • Añadir, modificar, eliminar socios.
  • Añadir, modificar, eliminar campaña de recibo recurrente o evento.
  • Crear, modificar, eliminar nuevos recibos recurrentes pendiente de pago de cada socio o sobre eventos puntuales.
  • Envíos automáticamente cada mes de notificaciones de recibos pendientes de pago a email de socio.
  • Envío de notificación a socio cuando recibo cambia estado a “pagado”.
  • Buscador que permite generar informes de tipo web y tipo .pdf filtrando por ID de recibo, intervalo de fechas, estado de pagos y forma de pago de recibos.
English:

Online web application developed in PHP and Mysql using Laravel framework. It allows you to manage invoices payments history and generate new recurring invoices of member’s club. Features:

  1. Administration panel with functionalities:
  • Add, modify, delete partners.
  • Add, modify, delete recurring campaign or event.
  • Create, modify, delete new recurring invoices pending payment from each member or on specific events.
  • Automatic sending each month of notifications of invoices pending payment to member email.
  • Sending notification to member when receipt changes status to “paid”.
  • Search engine that allows you to generate web-type and .pdf-type reports filtering by receipt ID, date range, payment status and receipt payment method.
