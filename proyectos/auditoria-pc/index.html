Auditoría de equipos informáticos

Desarrollo backend y frontend de aplicación web en PHP y Mysql utilizando el framework de Laravel. Objetivo: crear un sistema que permita administrar la auditoría de equipos informáticos de empresas clientes y generar reporte web y en fichero pdf de los equipos analizados. Características:

  1. Panel de administración con funcionalidades:
  • Panel de administración generado basado en CRUDs, gráficas Chart.js y plantilla Bootstrap AdminLTE.
  • Tres niveles de usuarios con privilegios diferenciados de acceso, creación y consulta de informes de auditoría (administrador, auditor, otro).
  • Crear/modificar/eliminar nueva cuenta de administrador/auditor.
  • Crear/modificar/eliminar opciones de componentes de equipos informáticos para auditar (Ej. CPU Intel Core I9 7800 Mhz) y asignar una puntuación de referencia.
  • Crear/modificar/eliminar empresas de clientes.
  • Crear/modificar/eliminar equipo informático de cliente a auditar.
  • Generar informe de auditoría de equipos informaticos de empresas clientes. Posibilidad de descargar en formato pdf.

Ver demo online (usuario: admin@admin.com - contraseña: admin)

English:

Backend and frontend development of web application in PHP and Mysql using the Laravel framework. Objective: create a system that allows to audit computer’s equipment of companies generating a web report and a pdf file of equipments analyzed. Characteristics:

  1. Administration panel with functionalities:
  • Administration panel generated based on CRUDs and Bootstrap AdminLTE template.
  • Three levels of users with differentiated privileges for access, creation and consultation of audit reports (administrator, auditor, other).
  • Create/modify/delete new administrator/auditor account.
  • Create/modify/delete hardware components to audit (eg Intel Core I9 7800 Mhz CPU) and assign a benchmark score.
  • Create/modify/delete companies.
  • Create/modify/delete companies computer equipment to be audited.
  • Generate audit report of computer equipment of companies. Possibility to download in pdf format.
